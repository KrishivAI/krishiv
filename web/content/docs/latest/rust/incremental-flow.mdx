<h2 id="overview">Overview</h2>
<p>`IncrementalFlow` maintains query results incrementally by processing `DeltaBatch` changes (weighted Arrow rows with an `Int64 _weight` column: `+1` = insert, `-1` = retract).</p>

<h2 id="construction">Construction</h2>
<div class="api-sig">IncrementalFlow::new(session: &amp;Session) -> Result&lt;IncrementalFlow&gt;</div>

<h2 id="methods">Methods</h2>

  <thead><tr><th>Method</th><th>Returns</th><th>Description</th></tr></thead>

    <tr><td>`register_source(name, schema)`</td><td>`Result&lt;()&gt;`</td><td>Register a named source that accepts DeltaBatch input.</td></tr>
    <tr><td>`register_view(name, query)`</td><td>`Result&lt;()&gt;`</td><td>Register a SQL query as an incrementally maintained view.</td></tr>

    <tr><td>`watch_output(view_name)`</td><td>`Result&lt;Receiver&lt;DeltaBatch&gt;&gt;`</td><td>Subscribe to incremental output deltas from a view.</td></tr>
    <tr><td>`checkpoint(path)`</td><td>`Future&lt;Result&lt;()&gt;&gt;`</td><td>Persist the current flow state to a checkpoint path.</td></tr>

<h2 id="example">Example</h2>

```rust
use krishiv_api::{Session, IncrementalFlow, Result};

#[tokio::main]
async fn main() -> Result<()> {
    let session = Session::embedded().await?;
    let mut flow = IncrementalFlow::new(&session)?;

    flow.register_source("orders", orders_schema)?;
    flow.register_view("totals",
        "SELECT customer_id, SUM(amount) AS total FROM orders GROUP BY customer_id")?;

    // Deliver a delta (weighted batch)
    let summary = flow.tick("orders", delta_batch).await?;
    println!("Processed {} rows, emitted {} delta rows", summary.rows_in, summary.rows_out);

    // Read current view
    let snapshot = flow.snapshot("totals").await?;
    Ok(())
}
```
