<h2 id="sessionbuilder">SessionBuilder</h2>
<p>All sessions are constructed via `SessionBuilder`. The builder is obtained from `Session::builder()` and follows the builder pattern.</p>
<div class="api-sig">Session::builder() -> SessionBuilder</div>

<h2 id="builder-methods">Builder Methods</h2>
<table class="api-table">
  <thead><tr><th>Method</th><th>Description</th></tr></thead>
  <tbody>
    <tr><td>`embedded()`</td><td>Shorthand for `SessionBuilder::default().build().await` in embedded mode.</td></tr>
    <tr><td>`mode(ExecutionMode)`</td><td>Set the execution mode: `Embedded`, `SingleNode`, `Distributed`.</td></tr>
    <tr><td>`coordinator_url(impl Into&lt;String&gt;)`</td><td>Flight/gRPC endpoint for distributed mode.</td></tr>
    <tr><td>`with_auth(impl AuthProvider)`</td><td>Attach a bearer-token or custom auth provider.</td></tr>
    <tr><td>`with_policy(impl PolicyHook)`</td><td>Attach a governance/access-control hook.</td></tr>
    <tr><td>`target_parallelism(NonZeroUsize)`</td><td>DataFusion target partition count for parallel execution.</td></tr>
    <tr><td>`with_iceberg_catalog(Arc&lt;KrishivCatalog&gt;, name)`</td><td>Register an Iceberg catalog under a given name (requires `iceberg-catalog` feature).</td></tr>
    <tr><td>`with_shuffle_partitions(Option&lt;u32&gt;)`</td><td>Override shuffle bucket count; `None` = auto.</td></tr>
    <tr><td>`config(key, value)`</td><td>Set a session config key.</td></tr>
    <tr><td>`build() -> Result&lt;Session&gt;`</td><td>Construct and return the session (async, requires `.await`).</td></tr>
  </tbody>
</table>

<h2 id="example">Example</h2>

```rust
use krishiv_api::{Session, Result};
use arrow::datatypes::{DataType, Field, Schema};
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<()> {
    let session = Session::embedded().await?;

    // Register in-memory data
    let schema = Arc::new(Schema::new(vec![
        Field::new("id",  DataType::Int64,   false),
        Field::new("val", DataType::Float64, true),
    ]));
    // ... build RecordBatch ...

    // SQL query
    let df = session.sql("SELECT id, val FROM my_table WHERE val > 10").await?;
    df.show().await?;
    Ok(())
}
```
