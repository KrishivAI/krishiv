<h2 id="overview">Overview</h2>
<p>Import from `krishiv.functions` or `krishiv.sql.functions`:</p>

```python
from krishiv.functions import col, lit, sum, avg, count, expr
from krishiv.sql.functions import upper, lower, date_trunc, coalesce
```


<h2 id="column-literal">Column and Literal</h2>
<table class="api-table">
  <thead><tr><th>Function</th><th>Signature</th><th>Description</th></tr></thead>
  <tbody>
    <tr><td>`col(name)`</td><td>`col(name: str) -> Column`</td><td>Reference a column by name.</td></tr>
    <tr><td>`column(name)`</td><td>`column(name: str) -> Column`</td><td>Alias for `col`.</td></tr>
    <tr><td>`lit(value)`</td><td>`lit(value: Any) -> Column`</td><td>Constant literal value (int, float, str, bool, None).</td></tr>
    <tr><td>`expr(sql)`</td><td>`expr(sql: str) -> Column`</td><td>Parse a SQL expression string into a Column.</td></tr>
    <tr><td>`call_function(name, *args)`</td><td>`Column`</td><td>Call a named SQL function or UDF with Column arguments.</td></tr>
    <tr><td>`function(name, *args)`</td><td>`Column`</td><td>Alias for `call_function`.</td></tr>
  </tbody>
</table>
