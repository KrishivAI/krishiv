<h2 id="queryresult">QueryResult</h2>
<p>Returned by synchronous `execute_local()` calls. Contains a list of collected Arrow batches.</p>
<table class="api-table">
  <thead><tr><th>Method</th><th>Returns</th><th>Description</th></tr></thead>
  <tbody>
    <tr><td>`batches() -> list[RecordBatch]`</td><td>`list`</td><td>Return the raw Arrow RecordBatch list.</td></tr>
    <tr><td>`row_count() -> int`</td><td>`int`</td><td>Total number of rows across all batches.</td></tr>
    <tr><td>`pretty() -> str`</td><td>`str`</td><td>Return a formatted ASCII table string.</td></tr>
    <tr><td>`show()`</td><td>`None`</td><td>Print the formatted table to stdout.</td></tr>
    <tr><td>`to_arrow() -> Table`</td><td>`pyarrow.Table`</td><td>Convert to a PyArrow Table.</td></tr>
    <tr><td>`to_pandas() -> DataFrame`</td><td>`pandas.DataFrame`</td><td>Convert to a pandas DataFrame (requires pandas).</td></tr>
    <tr><td>`len() -> int`</td><td>`int`</td><td>Same as `row_count()`; also works with `len(result)`.</td></tr>
  </tbody>
</table>

<h2 id="queryhandle">QueryHandle</h2>
<p>Returned by `execute_remote()` and `submit_async()`. Represents an in-flight asynchronous query.</p>
<table class="api-table">
  <thead><tr><th>Method</th><th>Returns</th><th>Description</th></tr></thead>
  <tbody>
    <tr><td>`query_id() -> str`</td><td>`str`</td><td>Unique query identifier.</td></tr>
    <tr><td>`status() -> str`</td><td>`str`</td><td>Current query status: "Running", "Completed", "Failed", "Cancelled".</td></tr>
    <tr><td>`is_done() -> bool`</td><td>`bool`</td><td>True if the query has completed (any terminal state).</td></tr>
    <tr><td>`progress() -> float`</td><td>`float`</td><td>Estimated progress in [0.0, 1.0].</td></tr>
    <tr><td>`collect() -> QueryResult`</td><td>`QueryResult`</td><td>Block until done and collect all results.</td></tr>
    <tr><td>`cancel()`</td><td>`None`</td><td>Request cancellation of the query.</td></tr>
  </tbody>
</table>

<h2 id="jobstatus">JobStatus</h2>
<p>Returned by `session.submit_stream_job()` and `session.jobs()`.</p>
<table class="api-table">
  <thead><tr><th>Method</th><th>Returns</th><th>Description</th></tr></thead>
  <tbody>
    <tr><td>`id() -> str`</td><td>`str`</td><td>Job ID.</td></tr>
    <tr><td>`name() -> str`</td><td>`str`</td><td>Job name.</td></tr>
    <tr><td>`state() -> str`</td><td>`str`</td><td>Job state: "Running", "Completed", "Failed", etc.</td></tr>
  </tbody>
</table>
