<h2 id="overview">Overview</h2>
<p>`DataFrame` is lazy â€” operations build a logical plan. Execution happens on `collect()`, `show()`, or `write_*` methods.</p>

<h2 id="transforms">Transformation Methods</h2>
<Callout type="info" title="Common methods"> `filter`, `select`, `group_by`, `agg`, `order_by`, `limit`, `join`, and `with_column` cover ~90% of batch transformations. Reach for `pivot`, `unpivot`, `sample`, and the rest only when you need them.</Callout>
