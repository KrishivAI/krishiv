Krishiv Platform is being developed as the workspace and control layer around
Krishiv Engine. Engine remains the compute layer for batch SQL, streaming, and
incremental view maintenance. Platform is intended to bring the surrounding
catalog, orchestration, governance, and operator experience into one place.

## Planned scope

- A web workspace for SQL, query history, and Iceberg catalog administration.
- Declarative data pipelines and scheduled SQL, Rust, and Python jobs.
- Team access controls, audit history, and governed MCP access.
- API and CLI surfaces for self-hosted operation around Krishiv Engine.

Krishiv Platform is source-available under the Business Source License 1.1 and
is intended for self-hosting. It is a separate product from the Apache-2.0
licensed Krishiv Engine.
