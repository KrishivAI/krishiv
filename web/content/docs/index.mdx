# Krishiv Docs

Krishiv is a Rust-native hybrid compute framework for batch SQL, streaming pipelines, and lakehouse-oriented data work.

This website is intentionally separate from the repository-level `docs/` directory. The existing `docs/` tree remains a development and engineering reference; public website content starts under `web/content/`.

## Start here

- [Getting started](/docs/getting-started)
- [Architecture concepts](/docs/concepts)
- [Runtime modes](/docs/runtime-modes)
- [Examples](/docs/examples)
- [Compatibility policy](/docs/reference/compatibility)

## Versioning model

The web app is prepared for release-branch documentation:

- `main` publishes the latest docs.
- `release/v0.1` can publish the `v0.1` docs.
- `web/versions.json` drives the version switcher and deployment metadata.
