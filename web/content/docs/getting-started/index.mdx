# Getting Started

Use this section to get from a clean checkout to a working Krishiv session.

## Recommended first path

1. Build the Rust workspace.
2. Run embedded mode locally.
3. Try batch SQL.
4. Try a streaming or DeltaBatch example.
5. Move to single-node or Kubernetes only after the local path is understood.
