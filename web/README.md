# Krishiv Web

Fumadocs/Next.js website for Krishiv public documentation, landing pages, blog,
changelog, examples, and versioned docs routing.

The repository-level `docs/` directory remains untouched and is kept for
engineering/development documentation. Public website content starts in
`web/content/` and can link back to canonical source files when needed.

## Commands

```bash
npm install
npm run dev
npm run build
npm run typecheck
```
